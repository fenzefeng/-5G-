## 介绍
  我们是来自山东大学的5人小队，成员包含：冯奋泽、刘希言、牟雅琳、齐安和、王兰齐。本项目是我们参加2024年智能车5G赛道时在蓝操场上拍摄的数据集，包含每一项赛道任务的数据原图dataset和处理后的图像output，希望对你有帮助。
